# Challenge Manager Platform

챌린지 매니저 플랫폼의 기획 문서와 현재 구현된 MVP 애플리케이션을 함께 관리하는 저장소입니다.

## 구성

- `챌린지_매니저_플랫폼_기획서.md`: 제품 기획서
- `챌린지_매니저_플랫폼_PRD_MVP.md`: MVP 범위 PRD
- `챌린지_매니저_플랫폼_PRD_구현착수용.md`: 구현 착수용 PRD
- `챌린지_매니저_플랫폼_프론트엔드_디자인_기획서.md`: 프론트엔드 디자인 기획서
- `challenge-manager/`: React + Vite + Express + SQLite 기반 MVP 앱

## 현재 앱 범위

- 관리자/참가자 인증
- 블로그/유튜브/틱톡 챌린지 관리
- 링크 및 이미지 기반 인증 제출
- 플랫폼별 링크 검증
- 참가자 랭킹/성공 판정
- 상금 정산 대상 및 지급 상태 관리
- CSV 다운로드 및 감사 로그

## 실행

앱 실행 방법과 주요 API는 [challenge-manager/README.md](./challenge-manager/README.md)에 정리되어 있습니다.
